# 핵심 기능 해결

> 필수 > ✅  
> 되면 한다 >  ⚠️

- 페르소나를 부어보면 어떨까? ✅
    
    - MBTI 페르소나 주입 하기는 편하지 않을까? ✅
        - mbti 궁합 차트
            
            ![image.png](attachment:5f2efb32-1e2c-4d3f-ba62-e2b1d0b157c0:image.png)
            
        - 자료가 명확해서 페르소나가 좀 더 명확하지 않을까?
            
        - 축에 대한 정보가 명확하기 때문에 mbti로 하는게 명확하지 않을까?
            
        - 가상 캐릭터 투입(레제 등)  ⚠️
            
- 화면에 캐릭터가 뜸 ✅
    
    - 캐릭터는 커스터마이징 해서 개인의 것을 사용할 수 있도록 ✅
    - 퀘스트를 깨면 해금 되도록 ⚠️
- 스크린샷을 찍을 수 있음 + 스크린샷을 통해서 해볼 수 있는 기능 ✅ (이미 검증됨 > 찍고 저장 가능)
    
    - 기능 단위로 쪼개기 ✅
        - 번역
        - 이미지 수정
        - 이미지에 대해서 묻기
    - 번역
    - 화면에 있는 정보를 띄워주는 것도 가능 할 것 같다.
    - 기능을 선택해서 실행 할 건지, 물어보고 시킬 건 지
    - 이미지를 수정 해줘?
    - 음성
- 클립 보드 ⚠️
    
    - 클립보드를 통해서 공용으로 사용 할 수 있으면 좋겠다.
- 멀티 요소 ⚠️
    
    - 멀티도 가능 할 것 같다.
    - 인터렉션 있으면 좋을 것 같다.
        - 이모지로 가능 하면 좋을 듯 > 밥 먹자 같은
        - 오늘 한 일 요약 보기
- 기타
    
    - 개인의 api 키 수정 가능 필요
    - 로그인 할 때 api key 받아오기
- 필수 기능 구현 > 테스트 해보고 > 추가 기능 구현
