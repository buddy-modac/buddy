    3. 페르소나 정하기

        2. 추천 퀴즈 받기 (클릭이 나을듯)

    3. 첫 대화 하면 해금
    4. …
5. 스샷 기능
    
    1. 스크린샷 전체/영역 저장

        2. 이미지 번역 > 텍스트 번역

            1. 주석을 단다거나
            2. 그림을 그린다거나
        4. 이미지 + 텍스트 추출본 보내서 번역된 이미지 받기

    1. 멀티
    2. 단독
