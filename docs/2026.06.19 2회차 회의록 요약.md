# **2회차 회의 요약**

## **정호님 합류 여부**

### **장점**

- 백엔드 역량 보강
- 새로운 시각 제공
- 해커톤 경험 보유

### **우려**

- 현재 기획 방향이 어느 정도 정리됨
- 컨텍스트 공유 비용 발생
- 추가 회의 필요
- 서버가 필수인 아이디어는 아님

### **결론**

- 화요일 사전 밋업 이후 결정
- 현재 방향에 공감한다면 합류 가능

---

# **추구미**

공통적으로

“확실한 핵심 기능 + 적당한 비주얼 완성도”

를 목표로 함.

- 기능이 먼저
- 프로토타입을 빠르게 만들기
- 비주얼은 평균 이상
- AI를 적극 활용
- 해커톤 기간 안에 완성 가능한 범위 유지

---

# **아이디어 방향**

초기 아이디어

1. 화면 상주형 AI 캐릭터
2. 화면 선택(Screen Capture) 기반 AI

### **결론**

두 아이디어를 결합

```
캐릭터 상주
→ 화면 선택
→ AI 분석
→ 말풍선 응답
```

# **현재 MVP 방향**

## **가장 유력**

“화면을 캡처하면 AI가 내용을 이해하고 캐릭터가 말풍선으로 답변해주는 데스크톱 AI 어시스턴트”

### **MVP**

1. 캐릭터 상주
2. 화면 캡처
3. AI 분석
4. 말풍선 응답
