# 1회차 요약

# **2회차 회의 참고**

## **논의된 아이디어**

### **1. 작업표시줄/화면 상주형 앱**

- TTS 기능
- 작업 표시줄 또는 화면에 항상 띄워두는 형태
- 참고 서비스: TaskHero
- Claude/Codex 작업 완료 시 알림을 주는 형태도 가능

### **2. 개발자 집중도 분석**

- 개발자의 컨텍스트 스위칭 분석
- CPU 프로파일링처럼 사용자의 집중 흐름 분석
- 어떤 앱이나 행동이 집중을 방해하는지 확인

### **3. 데스크톱 AI 캐릭터**

- 봉고캣처럼 화면에 상주
- 필요한 정보 표시
- 자주 사용하는 기능이나 매크로 실행
- AI 사용 중 겪는 불편함을 해결하는 방향 검토

### **4. 화면 선택(Screen Capture) 기반 AI**

- 화면 일부를 드래그해서 선택
- 번역
- 요약
- AI 질문
- 코드 분석

---

## **현재 방향**

### **후보 1**

작업표시줄/화면 상주형 서비스

### **후보 2**

화면을 선택해서 AI 기능을 사용하는 서비스

### **의견**

1번과 2번을 결합하는 방향이 괜찮아 보임

예시

- 화면에 AI 캐릭터 상주
- 단축키로 화면 영역 선택
- 번역/요약/질문 기능 제공
- 결과를 팝업 또는 캐릭터 말풍선으로 표시

### **AI 데이터 접근**
